  • Rank 500以上をEXP近似式で対応
  • 獲得アイテム一覧の表示
  • イベント期間外の利用時に対応 (警告を出す、現在時刻を次のイベント開始時刻にしてLP補完等)
  • 開催中のイベントをデフォルト値に設定
  • 獲得アイテムを石の価値に換算した際の石効率最適化
